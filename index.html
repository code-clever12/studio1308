Our voice is consistent everywhere — from Instagram captions to appointment reminders. Warm authority, never stiff. Aspirational, never out of reach.

Confident

Warm

✓ Do
  • —Warm, golden-hour or studio lighting

Design Rules

Do's & Don'ts

These rules protect the brand equity we've built. When in doubt, ask: does this feel refined and warm? If not, simplify.

✓ Always Do This
  • Use Cormorant Garamond for all headings and display text
  • Let cream and white space breathe around the burgundy — never crowd it
  • Use Champagne Gold as an accent only — sparingly, intentionally
  • Keep button text ALL CAPS in Montserrat 500
  • Maintain 1px borders — thin lines signal premium
  • Let hero images bleed full-width to the edges
  • Use lowercase Cormorant Garamond for section titles — feels intimate
  • Generous card padding — minimum 32px on all sides
✕ Never Do This
  • Never use burgundy as a large section background — reserve for accents and CTAs
  • Never use bold (600+) Cormorant Garamond — heaviness destroys the European feel
  • Never place gold and burgundy on the same interactive element
  • Never use sans-serif fonts for hero or section headings
  • Never center-align body copy that runs more than 3 lines
  • Never add drop shadows to text — use color contrast instead
  • Never use pure black (#000000) — always use Night Plum (#1C0912)
  • Never cram more than 3 font sizes into a single card component
